Symbolic regression (SR) is a core paradigm for discovering interpretable equations from data. However, progress in SR is increasingly constrained by fragmented research workflows, which hinder reproducible comparison and systematic extension. We present SREvoLab, a lightweight research infrastructure for symbolic regression. SREvoLab unifies 10 heterogeneous SR backends under a common execution interface with controlled environments and canonicalizes multiple benchmark families into a shared schema. It also supports repository onboarding, dataset homogenization, protocol compilation, and formula-aware validation within a review-first audited loop for bounded iterative improvement. We evaluate SREvoLab and find that formula validation on canonicalized datasets reaches split-level NMSEs as low as $10^{-15}$, while pilot archived runs provide aligned ID/OOD summaries across multiple methods and datasets. These results establish SREvoLab as a practical foundation for reproducible and extensible research in symbolic regression.
