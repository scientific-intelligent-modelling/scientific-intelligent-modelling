# 开题报告模板

这个目录提供一份可直接改写的中文 LaTeX 开题报告模板，默认文件为 `开题报告.tex`。

## 文件说明

- `开题报告.tex`：开题报告主体模板，已经预置常见章节结构。
- `references.bib`：BibTeX 文献库占位文件，可逐步补充参考文献。

## 建议使用方式

1. 先替换题目、姓名、学号、导师、学院等基础信息。
2. 按章节逐步补充研究背景、研究目标、技术路线、进度安排等内容。
3. 若需要插图，可将图片放到本目录或 `paper/figures/` 下，再在模板中取消对应图环境注释。

## 编译命令

```bash
cd paper/开题
xelatex 开题报告.tex
bibtex 开题报告
xelatex 开题报告.tex
xelatex 开题报告.tex
```

如果暂时还没有插入引用，`bibtex` 阶段出现空引用提示是正常的，后续补充 `\\cite{}` 和 `references.bib` 条目即可。
